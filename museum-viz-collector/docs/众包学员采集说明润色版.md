# 博物馆展陈可视化采集任务说明

## 一、项目背景

博物馆展览里，除了文物、展品和文字说明，常常还会出现地图、时间线、图表、信息图、互动屏、数字装置等内容。它们能帮助观众理解展览中的时间、地点、人物、事件、工艺流程和知识关系，也是展览讲故事的重要方式。

本次采集任务关注的就是这些“展陈中的可视化”。请各位学员选择一个博物馆展览进行参观，重点记录展览中用于解释信息、组织信息或帮助观众理解内容的可视化，并通过在线网站完成照片和文字记录。

这些采集结果会用于后续整理博物馆展陈可视化案例，帮助我们分析不同博物馆如何通过视觉方式呈现知识、组织展览，并引导观众理解内容。

## 二、采集任务范围

每位学员需要选择一个博物馆中的一个展览开展采集。一次采集任务对应一个展览。请注意，任务是否冲突看的是“展览”，不是“博物馆”：同一座博物馆可以有多位学员采集，但不能重复采集同一个展览。

采集时可以重点关注以下对象：

- 地图：历史地图、路线图、分布图、展区导览图等。
- 时间线：历史年表、人物生平线、事件发展线等。
- 图表：柱状图、饼图、折线图、比例图、统计图等。
- 流程图：制作流程、工艺流程、制度流程、考古过程等。
- 关系图：人物关系、机构关系、文化传播关系、知识结构图等。
- 信息图：图标、数字、文字、图片组合而成的信息说明。
- 数据墙：集中展示大量数字、年份、地点、名单或统计信息的墙面。
- 互动屏：触摸屏、查询屏、互动地图、数字沙盘等。
- 实体装置：通过模型、灯光、机械结构、地面图形等方式表达信息的装置。

一般不需要记录的对象包括：

- 单独一件文物或展品。
- 只有普通说明文字的展板。
- 没有信息解释作用的装饰图案。
- 只用于展示照片、但没有分类、时间、地点或关系组织的普通照片墙。

如果现场情况不确定，可以先记录下来，并在文字描述中说明你的判断依据。

## 三、采集的数据组织结构

本次采集任务以一个展览为单位。这个展览位于某一座博物馆中，所以需要先填写博物馆名称、地址和展览名称。进入展览后，再根据现场空间和内容，把展览划分成若干个“展厅单元”。每个展厅单元中，再记录其中出现的可视化项。

“展厅单元”可以理解为展览内部的一个相对完整的空间或主题部分。它可能是一个独立展厅，也可能是展览中的一个章节、一个区域或一段参观动线。划分展厅单元的目的，是为了说明每个可视化项具体出现在展览的哪个位置、处在什么样的展陈环境中。

## 四、基础信息填写要求

进入采集系统后，请先填写基础信息，包括：收集人姓名、收集人单位、联系方式、调研时间、所在城市、博物馆名称、博物馆地址、展览名称、展览举办时间、展览在线链接和展览介绍等。

填写要求：

- 博物馆名称和展览名称尽量使用现场正式名称。
- 展览举办时间如为常设展，可填写“常设展”。
- 展览在线链接可填写官网页面、公众号文章、购票页或其他公开介绍页面。
- 展览介绍不用太长，说明展览主题、主要内容和展品类型即可。

## 五、展厅单元填写要求

展厅单元是展览中相对完整的一个空间或主题部分。可以按现场展厅名称划分，比如“序幕厅”“今日北大展厅”；也可以按展览章节划分，比如“第一部分 起源”“第二部分 发展”。

划分时请注意：

- 不要把整个大型展览全部写成一个单元，否则后续很难定位具体内容。
- 不要把每个展柜都单独拆成一个单元，除非该展柜本身就是一个完整主题区。
- 如果现场已经有明确章节标题，优先使用现场标题。

每个展厅单元需要填写以下内容：

| 填写内容 | 填写要求 |
|---|---|
| 单元名称 | 优先使用现场展厅名、章节名或导览图上的名称；如果现场没有明确名称，可以根据空间位置或主题自行概括。 |
| 单元描述 | 写清楚这个单元在展览中的位置、空间环境和主要内容。可以说明它位于哪里、主要展示什么、观众从哪里进入和离开、现场灯光和人流如何，以及可视化内容主要出现在哪些位置。 |
| 环境照片 | 环境照片主要用来说明可视化项所在的空间位置，不是为了拍单个展品。建议拍单元入口或整体空间、主要展线方向，以及可视化项与展柜、展品、墙面、观众动线之间的关系；每个展厅单元建议拍 1-3 张。 |

## 六、可视化项填写要求

可视化项是本次采集的核心。每发现一个符合条件的可视化内容，请单独建立一条记录，并重点完成文字描述。文字描述可以直接口述转录，不需要写成正式文章，但最好按照下面四个角度说清楚。

| 描述角度             | 需要说明什么（不限于）                                                                                                                                                                                                                                                         |
| -------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| 可视化本身           | - 可视化类型：它是地图、时间线、柱状图、关系图、信息图、数据墙、互动屏，还是实体装置等。<br>- 内容主题：它主要在讲什么内容。<br>- 信息表达方式：它表达了哪些信息？用了位置、长度、颜色、大小等哪种方式来表达？                                                         |
| 可视化在展览中的功能 | - 新信息：相比旁边的文字说明或实物展品，它多提供了什么信息？<br>- 展览作用：它在展览中起什么作用，例如概览、导览、解释背景、展示关系、呈现变化、帮助比较等。                                                                                                                      |
| 和人的交互形式       | - 媒介和形态：它是静态、动态还是可交互的？是屏幕、投影、印刷品，还是实体装置？<br>- 观众行为：观众是看、触屏、走进去、动手操作，还是只是路过？<br>- 空间位置：它和周围展品是什么关系？位于参观动线的入口、中段、高潮处还是出口附近？         |
| 你对它的评价         | - 第一眼感受：它好不好看，是否有吸引力或冲击力。<br>- 是否易懂：能不能看懂，是否需要额外背景知识，是否成功传达了信息。<br>- 保存状态：有没有损坏、黑屏、褪色、失灵、缺损或内容过时等问题。<br>- 观众友好度：对儿童、普通观众或专业观众是否友好。 |


口述或填写时，可以直接按顺序说：“从可视化本身来看……；从它在展览中的功能来看……；从和人的交互形式来看……；从我的评价来看……”。四个角度不需要写得一样长，但请尽量多记录现场能观察到的信息。如果还有其他感受、想法或建议，也可以一起补充。

## 七、现场采集流程

建议按以下顺序完成采集：

1. 先在群内填写任务登记表，写明计划采集的博物馆和展览名称，确认该展览没有与其他同学重复。
2. 到馆后再次确认博物馆名称、地址和展览名称。
3. 进入该展览，先观察展览整体结构，判断可以分成几个展厅单元。
4. 进入每个展厅单元后，先拍环境照片，再填写单元名称和单元描述。
5. 在该单元内寻找可视化项。
6. 对每个可视化项依次拍摄位置照片、完整照片和细节照片。
7. 按“可视化本身、可视化在展览中的功能、和人的交互形式、你对它的评价”四个角度完成可视化项的文字描述。
8. 离开展厅单元前检查是否漏拍、漏填。
9. 完成全部展厅单元后，确认照片和文字记录已经整理完整。


采集信息不需要写得学术，但必须清楚、具体、真实。请尽量让没有去过现场的人，也能通过你的照片和描述判断这个可视化项的内容、位置、形式和展览作用。
